Login Location Notifier is a free plugin that empowers you to track your account access by email notification.

== Description ==

** Notify user with their account access location **

* Just install this plugin and track your account access location to your mailbox.
* If you want to set/changes your IP address go to this plugin setting and enter your current IP address.

== Installation ==

This section describes how to install the plugin and get it working.

In WordPress:

* Go to Plugins > Add New > search for :: Login Location Notifier ::
* Press Install Now for the Login Location Notifier plugin
* Press Activate Plugin

To install manually instead:

* Upload the login-location-notifier  directory to the /wp-content/plugins/ directory
* Activate the plugin through the Plugins menu in WordPress
